# AAItrade — Claude Code Context

## Project Overview
AAItrade (Autonomous Artificial Intelligence Trade) is an AI-powered autonomous stock trading system. Claude is the core decision-making brain. Everything else — data pipelines, memory, retrieval, integrations — exists to make Claude smarter at trading.

## Markets
- Indian markets: NSE, BSE
- US / international markets

## Phase
Currently in **Phase 1: Planning**. No code has been written yet. Architecture and feature decisions are being finalized.

## Goals
1. Learn Python, API building, and LLM integration through building this
2. Create a genuinely professional, scalable trading tool

## Key Principles
- Claude is the decision maker — not a rule-based system
- Minimize hallucination risk: ground every decision in real, retrieved data
- Risk management is non-negotiable — never expose full capital to a single position
- Build to learn first; optimize and scale second

## Working Style
- Phase 1 is discussion and planning only — no code until architecture is decided
- Prefer depth over speed; think through trade-offs before committing to a stack

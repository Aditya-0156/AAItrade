# AAItrade

**Autonomous Artificial Intelligence Trade** (sounds like *iTrade*)

An AI-powered autonomous stock trading system that handles analysis, decision-making, and execution for both Indian (NSE/BSE) and US/international markets.

## What It Does
You give it capital. It handles everything:
- Market data ingestion and analysis
- News and sentiment processing
- Trade decision-making (powered by Claude)
- Order execution
- Portfolio and risk management
